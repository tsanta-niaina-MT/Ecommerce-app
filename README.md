# Application E-commerce

Cette application est un projet e-commerce simple comprenant une partie front-end et back-end.  
Le back-end est développé avec **Node.js** et **Express**, et la base de données utilise **PostgreSQL**.  
Le front-end est une application web responsive utilisant **HTML**, **CSS (Bootstrap)**, et du **JavaScript** côté client.

---

## Fonctionnalités principales

- Liste des produits avec affichage dynamique
- Gestion d’un panier d’achat côté client (localStorage)
- Ajout, modification, suppression des articles dans le panier
- Passage de commande (checkout)
- Authentification utilisateur (connexion/inscription)
- Recherche de produits
- Interface responsive et moderne avec Bootstrap
- API REST sécurisée avec Express pour la gestion des produits, commandes et utilisateurs
- Base de données relationnelle PostgreSQL pour stocker les données produits, utilisateurs, commandes

---

## Technologies utilisées

- **Back-end :**
  - Node.js
  - Express.js
  - PostgreSQL
  - pg (driver PostgreSQL pour Node.js)
  - JWT (JSON Web Token) pour l’authentification
  - bcrypt pour le hash des mots de passe

- **Front-end :**
  - HTML5 / CSS3
  - Bootstrap 5
  - JavaScript (ES6+)
